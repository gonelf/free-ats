# KiteHR — 30-Day SEO Blog Content Plan

**How to use:** Check off each post as it's written. To request a post, just say:
> "Write today's blog post" or "Write Day [N]'s blog post"

Claude will write the full post ready to add to `src/app/blog/posts.ts`.

**Writing rules (apply to all posts):**
- Voice: practical, direct, no HR jargon — write for a founder or hiring manager, not a CHRO
- Primary keyword in H1, first 100 words, and at least two H2s
- Meta description: 150–160 chars with primary keyword + value statement
- Minimum 3 internal links per post using natural anchor text
- KiteHR feature mentions: natural 1–2 paragraphs in context, never a sales pitch
