# SOP Content Plan — Phases 2–6

This file drives the `generate-sop` cron job. Each entry below corresponds to one SOP that will be
auto-generated via Gemini Flash and stored in the `GeneratedSop` table. The cron picks the next
3–5 ungenerated entries every 2 days (in order), calls the AI, and publishes the result.

Phase 1 (30 SOPs) is hardcoded in `src/app/hr-sop/hr-sop-data.ts` and does NOT appear here.
